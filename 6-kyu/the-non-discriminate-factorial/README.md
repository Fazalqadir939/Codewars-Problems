<h2><a href=https://www.codewars.com/kata/53fe2171d5679bff300007d9/train/c target="_blank">The Non-Discriminate Factorial</a></h2><h3>6 kyu</h3><p>The factorial function is widely found in pure mathematics, most commonly in combinatorics and discrete probability. Typically the factorial is a function defined over nonnegative integers in such a way:</p><p>For <code>n=0</code>, <code>factorial(n) = 1</code>. For <code>n&gt;0</code>, <code>factorial(n) = n*(n-1)*...*2*1</code>.</p><p>For example</p><p><code>factorial(2) = 2</code></p><p><code>factorial(4) = 24</code>. </p><p>It might seem odd that <code>factorial(0) = 1</code>, but it is more of a convenience than anything else.</p><p>This problem requires you to define an extended version of the factorial function, one that accepts all integers! For nonnegative integers, your function should behave as above. </p><p>For <code>n&lt;0</code>, <code>factorial(n) = (n)*(n+1)*...*(-2)*(-1)</code>.</p><p>For example</p><p><code>factorial(-1) = -1</code></p><p><code>factorial(-2) = 2</code></p><p><code>factorial(-3) = -6</code></p><p>Get to it! Also, note that you will only get integers as input. No need for error messages:)</p>
